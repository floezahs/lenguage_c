Capítulo 01

Variables

Una variable es una caja con nombre donde el programa guarda un dato. Puedes cambiar su contenido en cualquier momento.

Concepto fundamental

¿Qué es una variable?

Imagina que tienes cajas físicas con etiquetas. int edad sería una caja etiquetada "edad" que solo puede guardar números enteros. Cuando escribes edad = 20, estás metiendo el número 20 dentro de esa caja.

En C debes decirle al compilador qué tipo de dato vas a guardar antes de usarla. Esto se llama declaración.

// tipo   nombre = valor inicial; int   edad = 20; // número entero float   precio = 19.99; // número decimal char   letra = 'A'; // un solo carácter double   masa = 9.109e-31; // decimal de alta precisión
Tipos de dato principales
int
4 bytes · −2,147,483,648 a 2,147,483,647
Números enteros sin decimales.
Uso: edades, contadores, puntajes.
int puntos = 100;
float
4 bytes · ~6 decimales
Números con parte decimal.
Uso: precios, temperaturas, notas.
float temp = 36.6;
char
1 byte · 0 a 127 (ASCII)
Un solo carácter entre comillas simples.
Uso: letras, símbolos.
char opcion = 's';
double
8 bytes · ~15 decimales
Decimal de doble precisión.
Uso: cálculos científicos, financieros.
double pi = 3.14159265358;

Capítulo 02

Constantes

Una constante es como una variable cuyo valor está bloqueado: se define una vez y nunca cambia. Ideal para valores fijos como Pi o el IVA.

Concepto fundamental

¿Por qué usar constantes?

Si escribes el número 3.14159 en 50 lugares de tu código y luego necesitas más precisión, tendrías que cambiar los 50. Con una constante PI solo cambias un lugar.

Además, el compilador te protege: si intentas modificar una constante por error, te avisa.

// Forma 1: #define (no ocupa memoria, el compilador lo sustituye) #define PI 3.14159 #define IVA 0.16 #define MAX 100 // Forma 2: const (recomendada, tiene tipo) const float pi = 3.14159; const int dias = 365; const double luz = 299792458.0; // m/s
⚠️  Error común: intentar modificar una constante.
const int MAX = 100; MAX = 200; // ❌ Error de compilación

Capítulo 03

Condicio­nales

Los condicionales le enseñan al programa a tomar decisiones: "si esto es verdad, haz X; si no, haz Y".

Estructura if / else if / else

C evalúa la condición dentro del paréntesis. Si es verdadera (≠ 0), ejecuta ese bloque y salta los demás. Si es falsa (= 0), pasa al siguiente.

if (nota >= 90) { printf("Excelente\n"); } else if (nota >= 70) { printf("Aprobado\n"); } else { printf("Reprobado\n"); }

Cuando tienes muchos casos con valores exactos, switch es más claro que una cadena de if/else:

switch (dia) { case 1: printf("Lunes\n"); break; case 2: printf("Martes\n"); break; case 6: case 7: printf("Fin de semana\n"); break; default: printf("Día laboral\n"); break; }

Capítulo 04

Ciclos

Los ciclos repiten un bloque de código automáticamente. Sin ellos, tendrías que copiar la misma línea cientos de veces.

Los tres tipos de ciclos en C
for
Cuando sabes exactamente cuántas veces repetir.
for (int i=0; i<5; i++) { printf("%d\n", i); }
while
Mientras una condición sea verdadera. Puede ser 0 veces.
while (n > 0) { n--; }
do-while
Como while pero se ejecuta al menos una vez.
do { scanf("%d", &op); } while (op != 0);

break y continue

break — detiene el ciclo

for (int i=0; i<10; i++) { if (i == 5) break; printf("%d ", i); } // Imprime: 0 1 2 3 4

continue — salta la iteración

for (int i=0; i<6; i++) { if (i == 3) continue; printf("%d ", i); } // Imprime: 0 1 2 4 5
